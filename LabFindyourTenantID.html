Find your Tenant ID

Find tenant ID through the Azure portal

  1. Sign in to the Azure portal.
  2. Select Azure Active Directory.
  3. Select Properties.
  4. Then, scroll down to the Tenant ID field. Your tenant ID will be in the box.

Find tenant ID with PowerShell

You can also find the tenant programmatically. To find the tenant ID with Azure PowerShell, use the cmdlet Get-AzTenant.

Azure PowerShell

Connect-AzAccount

Get-AzTenant

Find tenant ID with CLI

Any of these commands:

az login

az account list

az account tenant list

Find tenant ID with CLI for M365 (3rd party)

Pre-req: Install npm from here: Download | Node.js (nodejs.org)

The CLI for Microsoft 365 is distributed as an NPM package. To use it, install it globally using:

npm i -g @pnp/cli-microsoft365

or using yarn:

yarn global add @pnp/cli-microsoft365

Login to Microsoft 365/Office365

m365 login

Find tenant ID with Microsoft Graph

Open Graph Explorer

https://developer.microsoft.com/en-us/graph/graph-explorer

Sign-in

Enter this URL

https://graph.microsoft.com/v1.0/organization

Click Modify Permissions as you may need permissions to access this URL. Once it fetches the details, click on all Consent buttons.

Verify the Response Headers

Look into the “id” value.
